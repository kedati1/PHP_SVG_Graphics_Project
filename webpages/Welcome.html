SVG Grapghics Objective

This is an SVG Graphic, and SVG stands for Scalable Vector Graphic.
SVG's are designed to provide Vector graphics with the ability to be interacted with by code.
In this SVG, I have javascript make it look like the box is moving across the screen and once it reaches a certain point, it changes colour.
